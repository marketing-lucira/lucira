📊 Excel Essentials & Business Metrics

Learn VLOOKUP, INDEX-MATCH, Pivot Tables, and Business KPIs (Revenue, Profit, Margin, CAC, LTV).

1️⃣ Simple Explanation

Excel is the foundational data tool used in every enterprise. Before storing data in massive databases, business data lives in spreadsheets.

💡 Hinglish Note: Excel sabse pehle seekhna zaroori hai kyunki har company (Swiggy, Amazon, Banks) ke business teams pehle Excel mein data dekhte hain.

Day 5-14 — Module 2

📝 SQL Beginner to Advanced (Production SQL)

Master SELECT, JOINs, GROUP BY, CTEs, Window Functions (ROW_NUMBER, RANK, LEAD, LAG), and Query Optimization.

1️⃣ Simple Explanation & Theory

SQL (Structured Query Language) is the language used to extract, filter, and transform data stored in Relational Databases (MySQL, PostgreSQL, BigQuery).

💡 Hinglish Note: SQL Data Engineer ka sabse important weapon hai. Databases se crores of rows filter karke nikalne ke liye SQL queries likhte hain.

2️⃣ Advanced SQL Example — Window Functions & CTE

WITH monthly_sales AS ( SELECT customer_id, SUM(amount) AS total_spent, DENSE_RANK() OVER (ORDER BY SUM(amount) DESC) AS rnk FROM orders GROUP BY customer_id ) SELECT * FROM monthly_sales WHERE rnk <= 5;
